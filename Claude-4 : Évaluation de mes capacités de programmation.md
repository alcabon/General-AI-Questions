**Nouvelles features linguistiques :**
- **Modules (Java 9)** : Project Jigsaw, module-info.java
- **var keyword (Java 10)** : inférence de types locaux
- **Switch expressions (Java 14)** : yield, pattern matching
- **Records (Java 14/16)** : classes de données immutables
- **Text blocks (Java 15)** : strings multi-lignes
- **Pattern matching (Java 17+)** : instanceof patterns, switch sur types
- **Virtual Threads (Java 21)** : Project Loom

**Écosystème moderne :**
- **Spring Boot 3.x** : annotations modernes, reactive programming
- **Gradle/Maven** : builds optimisés, multi-modules
- **Microservices patterns** : Spring Cloud, circuit breakers
- **Reactive streams** : WebFlux, R2DBC
- **Native compilation** : GraalVM, Spring Native

**Outils modernes :**
- **IDEs** : IntelliJ features avancées, refactoring
- **Testing** : JUnit 5, Testcontainers, Mockito moderne
- **CI/CD** : GitHub Actions, containerisation Docker
